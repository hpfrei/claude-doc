clairview sits between Claude Code and the LLM, giving you a browser dashboard over everything. Zero cloud infrastructure.

  • All Claude Code traffic proxied and recorded — every API call, every response, fully transparent
  • LLM switchable between Anthropic (default), OpenAI, Gemini, DeepSeek, and Kimi — one dropdown, automatic protocol translation
  • User interactions intercepted — AskUserQuestion, confirmations, and prompts appear in the browser and answers flow back to Claude Code
  • Inspector shows text interactions, tool calls, MCP calls, hooks — with token counts, timing, and full payloads
  • Workflow steps can use different models per step, with configurable context piped from previous steps

Everything in one dashboard

Wire-level API inspection

See every request, response, SSE event, and tool call between Claude Code and the LLM. Token usage, timing, headers — nothing hidden.

Route through any model

Send Claude Code traffic through OpenAI, Gemini, DeepSeek, Moonshot, or local Ollama models. Automatic protocol translation. Switch with a dropdown.

NL-to-workflow pipelines

Describe a workflow in plain English, get a compiled multi-step pipeline. Branching, parallelism, retries, per-step profiles. Each step is a full claude -p session.

Custom tools from the browser

Form-driven editor for MCP tools. Define parameters, write the handler, test inline. Auto-generated server — Claude gets the capability instantly.

Multi-tab Claude sessions

Run Claude Code from any device. Multiple parallel sessions, each with its own profile and working directory. AskUserQuestion interception, live task tracking.

Profiles, skills, agents, hooks

Named capability bundles control model, effort, permissions, budget. Create skills, agents, and lifecycle hooks from a visual editor. Stored in .claude/ for auto-discovery.

How it works

Running in 60 seconds

1
git clone https://github.com/hpfrei/clairview.git && cd clairview
2
npm install
3
npm start # open localhost:3457
Inspect external sessions Point any Claude Code instance at the proxy:
ANTHROPIC_BASE_URL=http://localhost:3456 claude -p "your prompt"
